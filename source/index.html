FAQ

Welcome to team Cool Beans' digital Bullet Journal! This BuJo was designed to help you organize tasks and events and stay on top of deadlines.

What is a Bullet Journal?

A traditional Bullet Journal, or BuJo, is a method of personal organization that incorporates scheduling, reminders, to-do lists, and other tasks into a single notebook. These items are written down as bulleted lists, hence the name "Bullet Journal." Digital BuJos are virtual versions of these notebooks.

How to Use This Bullet Journal

First, you can click on the logo in the top left of the header ( ) to pick your favorite color theme. This theme will stick even if you refresh the page!

Then, notice that the default view in our Bullet Journal is a 3-column display with sections for tasks you're currently working on (In Progress), tasks that you haven't started yet (Backlog), and tasks that you've completed (Complete). You can move bullet points between these sections as necessary using the complete ( ) and migrate ( ) buttons. Simply hover the cursor over the bullet point you want to move and these buttons will appear.

You can choose to move tasks that have been completed into the Archive with the archive button ( ); otherwise, if a task has been completed for a while, we'll automatically archive it for you! You can see all the bullet points that have been archived by clicking the Archive toggle in the header ( ).

Alternatively, you can delete a bullet point by hovering over it and clicking the edit button ( ) and then clicking the button that reads "Delete Bullet". All of the bullet points in the Archive can be deleted at once by clicking the delete button by the section's header ( ). Similarly, all of the bullet points in your BuJo can be deleted at once by clicking the clear-all button ( ). Note that this will also reset the color theme to the default!

You can open the editor to create new bullet points from the 3-column view by clicking the editor button in the bottom right ( ). You can also edit existing bullet points by hovering over them and clicking the edit button ( ), making the necessary changes, and clicking the button that reads "Save Changes".

Finally, you can search for bullet points with a specific date or label by clicking the search button ( ).
